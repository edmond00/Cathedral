# sections

- **Summit Approach** — steep, exposed, difficult traversal
- **Exposed Summit** — wind-scoured, open sky, extreme conditions

# areas

Pick 2–3 from:
- Ridge (narrow exposed spine)
- Summit Plateau (flattish top — rounded peak variant)
- Windswept Ledge (jutting shelf, dramatic view)
- Ice Shelf (very high peaks only — atmospheric, impassable)
- Scree Gully (steep loose-rock channel)

# spots

**Rock/Ice:**
- Cairn (stone pile left by past climbers) → Stone, Flint
- Wind-Carved Rock → Stone ×2, Lichen
- Crevice → Flint, Bat Roost (shallow NPC marker)
- Ice Formation → (atmospheric only, no items)

**Rare Vegetation:**
- Sheltered Hollow → Gentian, Valerian (rare herbs, only in sheltered spots)
- Lichen Crust → Lichen

# roads/doors

Connections within the peak:
- Ridge Path: Summit Approach ↔ Ridge
- Summit Path: Ridge ↔ Summit Plateau or Windswept Ledge
- Gully: Ridge ↔ Scree Gully
- Cliff Descent: Summit Approach → Mountain (CLIMB_DOWN — primary exit, high difficulty)

# npc

**Human:** None. Peak is uninhabited.

**Shallow:**
- Eagle (always present — peak is eagle territory)
- Snow Hare (rare — 20%)
- Raven (common)

**Beast:**
- Ibex / Mountain Goat (30%)
- Wolf (10% — very rare)

# items

**Stone/mineral:**
- Stone, Flint, Lichen

**Rare herbs (reward for difficult traversal):**
- Gentian, Valerian

**Animal drops:**
- Eagle Feather, Snow Hare Pelt (rare), Wolf Pelt (very rare)

# comments

**RNG rules:**
- Peak always accessed from Mountain via cliff connection — never directly from plain or forest
- Summit shape: pointed (ridge + single ledge) vs rounded (plateau + multiple ledges)
- Ice Shelf only on highest peaks (world-level altitude flag) — atmosphere only
- Cairn present in ~60% of peaks (implies past human passage, but no residents)
- Very few spots total — peak should feel sparse and exposed
- Eagle always spawns, non-negotiable

**Tone note:**
Austere and remote. Minimal items, minimal NPCs. Reward is the view and the rare herbs, not loot density.
