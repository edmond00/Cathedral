# sections

- **Flatlands** — open, low-lying ground; wide sky; long sightlines
- **Highlands** — gentle rises and shallow valleys
- Optional variant section: **Wetlands** — boggy ground, reeds, willows (replaces one section in ~25% of plains)

# areas

Pick 3–5 from:
- Grassland (open, wind-swept)
- Meadow (wildflowers, sheltered)
- Heath (dry scrub, heather, gorse)
- Hill (elevated, exposed)
- Valley (damp hollow, sheltered)
- Boggy Ground (wetland variant only)
- Hedgerow (boundary strip)

# spots

**Trees (1–2 per area, species by area type):**
- Oak → Acorn, Branch, Bark
- Ash → Branch, Bark
- Hawthorn → Hawthorn Berry, Branch
- Willow → Branch, Bark (wetland only)
- Elder → Elderberry, Branch

**Ground features:**
- Boulder → Rock, Flint, Lichen
- Flower Patch → Daisy, Poppy, Clover, Thistle, Heather (pick 2)
- Berry Bush → Blackberry *or* Bilberry *or* Sloe
- Mushroom Ring → Mushroom ×2
- Reed Bed → Reed, Clay (wetland only)
- Rabbit Burrow → (shallow NPC spawn marker, no items)

# roads/doors

Paths connecting areas within the plain:
- Open Path: Grassland ↔ Meadow
- Open Path: Meadow ↔ Hill
- Open Path: Hill ↔ Valley
- Bog Track: Grassland ↔ Boggy Ground (wetland variant)
- Hedge Gap: Hedgerow ↔ Grassland
- Heath Track: Heath ↔ Hill

# npc

**Human:** None by default. Plain is uninhabited terrain.

**Beast:**
- Boar (hostile, 25%)
- Fox (non-hostile, 40%)
- Deer (shy, 50%)
- Wolf (hostile, rare — 10%)

**Shallow:**
- Hare, Rabbit (common)
- Crow, Lark, Sparrow (always some birds present)
- Frog, Toad (wetland variant)

# items

**Forage:**
- Blackberry, Bilberry, Sloe, Hawthorn Berry, Elderberry, Acorn, Beechnut, Mushroom
- Apple, Pear (only if apple/pear tree spawned)

**Flora:**
- Daisy, Poppy, Clover, Thistle, Heather, Nettle, Reed

**Stone/mineral:**
- Rock, Flint, Lichen, Clay (wetland)

**Animal drops:**
- Feather (birds), Deer Hide, Boar Tusk, Wolf Pelt

# comments

**RNG rules:**
- Core identity drawn first: open plain vs heath vs wetland (affects section and area pool)
- Wetland variant swaps one section for Boggy Ground, adds Reed Bed spots and willow trees
- Tree species: open plains get oak/hawthorn, heath gets elder/hawthorn, wetland gets willow/elder
- Berry bush type randomised per bush instance
- Bird species always present but random which type dominates
- Wolf only appears if forest location is adjacent in the world map
- No human NPCs — plain is wild, uninhabited terrain
